# Fraud Detection System (Flask + ML)

A Flask-based web application for real-time fraud detection using machine learning.

## Features
- ML-based fraud probability prediction
- One-hot encoding & scaling pipeline
- Risk classification (Low / Medium / High)
- Transaction history tracking
- Explainable risk insights

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy
- HTML/CSS

## How to Run
```bash
pip install -r requirements.txt
python app.py
